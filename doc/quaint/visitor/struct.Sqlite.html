Struct quaint::visitor::Sqlite

source · [−]
pub struct Sqlite<'a> { /* private fields */ }
Expand description

A visitor to generate queries for the SQLite database.

The returned parameter values implement the ToSql trait from rusqlite and can be used directly with the database.
