insert_emulated_on_delete

Inserts emulated referential actions for onDelete into the graph. All relations that refer to the model row(s) being deleted are checked for their desired emulation and inserted accordingly. Right now, supported modes are Restrict and SetNull (cascade will follow). Those checks fail at runtime and are inserted between parent_node and child_node.
