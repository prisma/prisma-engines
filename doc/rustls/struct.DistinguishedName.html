Expand description

A DistinguishedName is a Vec<u8> wrapped in internal types.

It contains the DER or BER encoded Subject field from RFC 5280 for a single certificate. The Subject field is encoded as an RFC 5280 Name. It can be decoded using x509-parser’s FromDer trait.

ⓘ
for name in distinguished_names {
    use x509_parser::prelude::FromDer;
    println!("{}", x509_parser::x509::X509Name::from_der(&name.0)?.1);
}
